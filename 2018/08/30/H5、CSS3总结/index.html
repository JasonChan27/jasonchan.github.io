H5、CSS3总结

一、HTML5

1、新特性

  • 用于绘画的Canvas元素
  • 用于媒介回放的video和audio元素
  • 对本地离线存储的更好的支持
  • 新的特殊内容元素，如：article、footer、header、nav、section
  • 新的表单控件，如：calendar、date、time、email、url、search

2、H5拖放

步骤：

  1. 设置元素为可拖放。draggable="true"
  2. 拖动什么。ondragstarrt和setData()
  3. 放到何处。ondragover(ev.preventDefault())
  4. 进行放置。ondrop和getData()(ev.preventDefault())

3、Canvas

Canvas拥有多种绘制路径、矩形、图形、字符及添加图像的方法。

步骤：

  1. 创建Canvas元素
  2. 通过javascript控制
    1. i、JS通过id寻找canvas元素
    2. ii、创建context对象
    3. ii、通过对象的方法绘制图形
  3. 理解坐标，如：fillRect(0,0,150,75)是在画布上绘制150x75的矩形，从左上角开始(0,0)

4、svg

  • 可伸缩矢量图形
  • 用于定义用于网络的基于矢量的图形
  • 使用XML格式定义图形

5、地理定位

  • 检测是否支持地理定位
  • 如支持，运行getCurrentPosition()，如不支持，向用户提示消息
  • getCurrentPosition()运行成功，向showposition中规定的函数返回一个coordinates对象
  • showPosition()函数获得并显示经度和纬度
  • getCurrentPosition(showPosition,showError)
    • pemission denied 用户不允许地理定位
    • position unavailable 无法获取当前位置
    • Timeout 操作超时
  • 在地图中显示结果：将经纬度传入谷歌或百度地图的api
    • watchPosition() 返回用户当前位置，并继续返回用户移动时的位置
    • clearWatch() 停止watchPosition()方法

6、web存储

  • localStorage(无时间限制)
    • localStorage.setItem(“lastname”,”Chen”)
    • localStorage.getItem(“lastname”)
    • localStorage.pagecount
  • sessionStorage(针对一个session进行数据存储，窗口关闭后，数据会被删除)
    • sessionStorage.clickcount
    • sessionStorage.pagecount

7、Manifest

  • 文件扩展名：.appcache
  • 在<hmtl>标签中包含此属性
  • 分三部分，如下：
    • CACHE MANIFEST
      　　＃ 2012-2-21 v1.0.0
      　　theme.css
      　　logo.gif
      　　main.js
    • NETWORK
      　　＃ 2012-2-21 v1.0.0
      　　theme.css
      　　logo.gif
      　　main.js
    • FALLBACK
      　　/html5/ /404.html (第一个是URI资源，第二个是替补)

8、web worker

运行在后台的JS

步骤：

  1. 创建web worker文件
  2. 创建web worker对象(w=new worker(“demo.js”))
  3. 终止web worker(w.terminate())
  • postMessage()向HTML页面传回一段消息
  • onmessage事件监听器(event.data)

9、服务器发送事件server-sent event

网页自动获取来自服务器的更新

步骤：

  1. 创建EventSource对象，发送更新页面的URI(demo.php)
  2. 每接收一次更新，就会发生onmessage事件
  3. 将event.data更新到指定id的元素中

10、input类型

  • email
  • url
  • number
  • range
  • date
    • type=date 选取日、月、年
    • type=month 月、年
    • type=week 选取周和年
    • type=time 选取时间(小时和分钟)
    • type=datetime 选取时间、日、月、年(UTC时间)
    • type=datetime-local 选取时间、日、月、年(本地时间)
  • search
  • tel
  • color

11、表单元素

  • datalist
    - 是输入域的选项列表
    - datalist内的option元素创建的
    - datalist绑定到输入域，用输入域的list属性引用datalist的id
  • keygen
    • 密钥对生成器，私钥(private key)存储于客户端，公钥(public key)被发送到服务器，公钥可用于之后验证用户的客户端证书
  • output

12、新表单属性

  • autocomplete，适用于：
    • form
    • input：text,search,url,telephone,email,password,datepickes,range,color
  • form,适用于所有input类型，必须引用所属表单的id
  • 表单重写属性
    • formaction
    • formencype
    • formmethod
    • formnovalidate
    • formtarget
  • height和width
  • list，规定输入域的datalist，datalist是输入域的选项列表
  • min、max、step，适用于：
    • datepickers
    • number
    • range
  • multiple，适用于：
    • email
    • file
  • novalidate
  • pattern
  • placeholder
  • required

二、CSS3

1、边框

  • border-radius
  • box-shadow，如：10px 10px 5px #888888
  • border-image，如：url(a.png) 30 30 round,参数分别为图片路径、图片边框向内偏移量、边框宽度、铺满方式。

2、背景

  • background-size
  • background-roigin,属性有：
    • border-box
    • padding-box
    • content-box
  • 多重背景图片，如：background-image:url(a.gif),url(b.gif)

3、文本效果

  • text-shadow,如：5px 5px 5px #FF0000，参数分别为水平阴影、垂直阴影、模糊距离、颜色
  • 自动换行,如：word-wrap:break-word
  • text-outline，文本轮廓
  • text-overflow，文本益处

4、字体

@font-face{
font-family
src
font-stretch
font-style
font-weight(100-900/normal/bold)
unicode-range
}

5、2D转换

  • translate(x,y)
  • translateX(n)
  • translateY(n)
  • scale(x,y)
  • rotate(angle)
  • skew(x-angle,y-angle)
  • matrix(n,n,n,n,n,n)，把所有转换方法组合在一起，需要六个参数，包含数学函数，允许旋转、缩放、移动及倾斜元素。

属性：

  • transform
  • transform：设置元素的基点位置

6、3D转换

  • translate3d(x,y,z)
  • translateX(x)
  • translateY(y)
  • translateZ(z)
  • scale3d(x,y,z)
  • rotate3d(x,y,z,angle)
  • perspective(n)
  • matrix(…),16个参数，4x4矩阵

属性：

  • transform
  • transform-origin
  • transform-style
  • perspective
  • perspective-origin
  • backface-visibility

7、过渡

  • 规定希望把效果添加到哪个css属性上
  • 规定效果的时长

例：transition:width 1s linear 2s;参数分别为css属性、过渡时间、过渡时间曲线(默认ease)、过渡何时开始。
transition:width 1s,-webkit-transform 2s;

8、动画

是使元素从一种样式逐渐变化为另一种样式的效果

  • 用百分比规定变化发生的时间，”from”、”to”等同于0%、100%
  • @keyframs规则用于创建动画

9、多列

  • column-count，元素被分隔的列数
  • column-gap，列之间的间隔
  • column-rule，列之间的宽度、样式和颜色规则

例：column-rule:3px outset #ff0000;

10、用户界面

  • resize,规定是否可由用户调整元素尺寸，如：div{resize:both;overflow:auto}
  • box-sizing,以确切的方式定义适应某个区域的具体内容，属性有：
    • content-box(只计算内容)
    • border-box(计算带内边距和边框)
    • inherit
  • outline,对轮廓进行偏移(轮廓不占用空间)，如：div{border:2px solid black;outline:20px solid red;outline-offset:15px;}
